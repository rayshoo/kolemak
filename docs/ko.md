# Kolemak

Windows에서 두벌식 한글 입력을 지원하는 Colemak 자판 드라이버입니다.

## 특징

- **설치 불필요** — `.exe` 파일 하나만 다운로드하여 실행하면 바로 사용 가능합니다. 별도의 드라이버 설치나 OS 레이아웃 변경이 필요 없습니다.
- **한글 IME 자동 감지** — Microsoft 한글 입력기의 한/영 상태를 자동으로 감지하여 두벌식 한글 입력 시 올바른 키를 전달합니다.
- **즉시 QWERTY 전환** — `Win+Space`로 언제든 QWERTY로 전환할 수 있어, 다른 사용자와 컴퓨터를 공유할 때 편리합니다.
- **CapsLock을 Backspace로** — CapsLock 키가 Backspace로 동작합니다. 기존 CapsLock 기능은 `Shift+CapsLock`으로 사용할 수 있습니다.

## 요구 사항

- Windows 10 이상
- Microsoft 한글 입력기 (Windows 기본 한국어 입력기)

## 빌드 변형

Colemak과 두벌식 한글 간의 `;` / `ㅔ` 키 충돌을 처리하는 두 가지 빌드를 제공합니다:

| 빌드 | 파일 | 영문 `;` 위치 | 한글 `ㅔ` 위치 |
|------|------|-------------|--------------|
| **Changed** | `kolemak-changed.exe` | Colemak `;`이 `P` 키에 위치 | `ㅔ`가 `P` 키로 이동 |
| **Unchanged** | `kolemak-unchanged.exe` | Colemak `;`이 `P` 키에 위치 | `ㅔ`가 기존 `;` 키에 유지 |

Colemak 배열에서는 QWERTY의 `P` 키가 `;`으로 바뀝니다. 두벌식에서 `P` 키는 `ㅔ`이므로 충돌이 발생합니다.

- **Changed** — `ㅔ`를 Colemak의 `;` 위치(`P` 키)로 이동합니다. 영문과 한글 모두 통일된 레이아웃을 원할 때 선택하세요.
- **Unchanged** — `ㅔ`를 원래의 물리적 키(`;` 키)에 유지합니다. 한글 입력을 기존 QWERTY 키보드와 동일하게 사용하려면 선택하세요.

## 설치

### 1. 다운로드

[Releases](https://github.com/rayshoo/colemak_korean/releases) 페이지에서 원하는 빌드를 다운로드합니다:
- `kolemak-changed.exe` 또는 `kolemak-unchanged.exe`

### 2. 파일명 변경 및 배치

다운로드한 파일의 이름을 `kolemak.exe`로 변경하고, 고정된 위치에 배치합니다 (예: `C:\Program Files\Kolemak\kolemak.exe`).

### 3. 환경 변수 설정 (선택)

`kolemak.exe`가 위치한 디렉토리를 시스템 PATH 환경 변수에 추가합니다:

1. `Win` 키를 누르고 **환경 변수**를 검색하여 **시스템 환경 변수 편집**을 엽니다
2. **환경 변수** 버튼을 클릭합니다
3. **시스템 변수**에서 `Path`를 선택하고 **편집**을 클릭합니다
4. **새로 만들기**를 클릭하고 디렉토리 경로를 추가합니다 (예: `C:\Program Files\Kolemak`)
5. 모든 대화 상자에서 **확인**을 클릭합니다

### 4. 작업 스케줄러 등록 (권장)

Windows 시작 시 관리자 권한으로 자동 실행되도록 설정합니다:

1. `Win` 키를 누르고 **작업 스케줄러**를 검색하여 실행합니다
2. **작업 만들기**를 클릭합니다 ("기본 작업 만들기"가 아님)
3. **일반** 탭:
   - 이름: `Kolemak`
   - **가장 높은 수준의 권한으로 실행** 체크
4. **트리거** 탭:
   - **새로 만들기** → 작업 시작: **로그온할 때** → **확인**
5. **동작** 탭:
   - **새로 만들기** → 동작: **프로그램 시작**
   - 찾아보기에서 `kolemak.exe`를 선택 → **확인**
6. **설정** 탭:
   - **다음 시간 이상 작업이 실행되면 중지** 체크 해제
7. **확인**을 클릭하고 작업 스케줄러를 닫습니다

## 사용법

### Colemak / QWERTY 전환

<kbd>Win</kbd>+<kbd>Space</kbd>를 눌러 Colemak과 QWERTY 레이아웃을 전환합니다. 전환 시 화면 좌측 상단에 현재 레이아웃 상태가 잠시 표시됩니다.

### CapsLock

| 단축키 | 동작 |
|--------|------|
| `CapsLock` | Backspace |
| `Shift+CapsLock` | CapsLock 토글 |

## 키 매핑

Colemak 모드에서 다음 QWERTY 키가 재매핑됩니다. 그 외의 키는 변경되지 않습니다.

| QWERTY | Colemak | 한글 (패스스루) |
|--------|---------|---------------|
| `E` | `F` | `ㄷ` / `ㄸ` |
| `R` | `P` | `ㄱ` / `ㄲ` |
| `T` | `G` | `ㅅ` / `ㅆ` |
| `Y` | `J` | `ㅛ` |
| `U` | `L` | `ㅕ` |
| `I` | `U` | `ㅑ` |
| `O` | `Y` | `ㅐ` / `ㅒ` |
| `P` | `;` | [빌드 변형](#빌드-변형) 참조 |
| `S` | `R` | `ㄴ` |
| `D` | `S` | `ㅇ` |
| `F` | `T` | `ㄹ` |
| `G` | `D` | `ㅎ` |
| `J` | `N` | `ㅗ` |
| `K` | `E` | `ㅏ` |
| `L` | `I` | `ㅣ` |
| `;` | `O` | [빌드 변형](#빌드-변형) 참조 |
| `N` | `K` | `ㅜ` |

## 알려진 제한 사항

- **키 인터셉트 방식** — Kolemak은 AutoHotkey를 통해 키 입력을 가로채어 변환하는 방식으로, OS 드라이버 수준에서 동작하지 않습니다. CPU 부하가 높거나 빠르게 연타할 경우 간헐적으로 QWERTY 키가 입력될 수 있습니다.
- **앱 호환성** — 일부 애플리케이션이나 입력 필드에서 재매핑된 키가 동작하지 않을 수 있습니다. 이 경우 `Win+Space`로 QWERTY로 전환하여 사용하세요.
- **Microsoft 한글 입력기 전용** — 서드파티 IME는 지원되지 않습니다. Windows 입력 방식이 Microsoft 한글 입력기로 설정되어 있어야 합니다.
