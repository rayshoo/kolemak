# Kolemak

A native Windows IME that provides Colemak keyboard layout with Korean Dubeolsik input support. Built on Windows TSF (Text Services Framework) for reliable integration with all applications.

Windows TSF 기반의 네이티브 입력기로, Colemak 자판 배열과 두벌식 한글 입력을 지원합니다. 모든 애플리케이션에서 안정적으로 동작합니다.
